1. **Подготовка к выполнению**.

1.1 Установить Java 17

1.2 Выполните команду

`chcp 65001`

перед запуском тестов средствами Maven. Это переключит кодировку консоли на UTF‑8.

1.3 Для корректного использования Lombok в среде разработки, необходимо установить _Lombok Plugin_

2. **Запуск тестов**

Для запуска тестов необходимо выполнить

`mvn clean test`

3. **Allure отчёты**

3.1 Для отображения отчётов в браузере, необходимо выполнить команду

`mvn allure:serve`

3.2 Пункт из ТЗ "Предполагайте, что отчет ваших автотестов будет читать человек, не связанный с программированием".
Открытие отчётов с описанием тестов можете найти во вкладке _Behaviors_ в окне браузера с отчётами
