1. **Подготовка к выполнению**.

1.1 Установить Java 17

1.2 Выполните команду

`chcp 65001`

перед запуском тестов средствами Maven. Это переключит кодировку консоли на UTF‑8.

1.3 Для корректного использования Lombok в среде разработки, необходимо установить _Lombok Plugin_

2. **Запуск тестов**

Для запуска тестов необходимо выполнить

`mvn clean test`

3. **Allure отчёты**

3.1 Для отображения отчётов в браузере, необходимо выполнить команду

`mvn allure:serve`

3.2 Пункт из ТЗ "Предполагайте, что отчет ваших автотестов будет читать человек, не связанный с программированием".
Открытие отчётов с описанием тестов можете найти во вкладке _Behaviors_

4. **Доп.информация**

Параллелизация части авто-тестов привела к тому, что логи выводятся не в нужном окне консоли.
В данный момент удалось записывать логи тестов в файл, содержащий имя теста по пути target/test-logs/test-${testName}.log.  
В них записываются логи slf4j, но не пишутся логи Rest Assured. Нахожусь в процессе фикса этой проблемы. 
В связи с этим в проекте существуют некоторые файлы, зависимости и аннотации, которые нужны для решения текущей проблемы.
